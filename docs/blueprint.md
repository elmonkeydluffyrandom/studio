# **App Name**: BibliaDiario

## Core Features:

- Dashboard View: Display recent entries in a card-based list, sorted by date.  Includes a search bar for filtering entries by keywords or tags.
- SOAP Entry Editor: Clean form for creating new entries with fields for Bible verse, observation, teaching, practical application, and tags.
- Verse Autocompletion: AI powered verse autocompletion based on Reina Valera 1960 Bible version using a retrieval tool
- Tag System: System to add tags to entries (e.g., #Prayer, #Family).
- PDF Export: Floating action button to export the current entry as a formatted PDF for printing.
- Firestore Integration: Store each entry as a document in Firestore, including creation timestamp.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke peace and reflection.
- Background color: Light gray (#F0F4F8), a desaturated version of the primary, for a clean and unobtrusive feel.
- Accent color: Light Purple (#BDB2FF) for interactive elements.
- Body and headline font: 'Alegreya' for a readable, elegant, intellectual feel.
- Use simple, minimalist icons for navigation and actions.
- Responsive design for comfortable use on mobile devices.
- Subtle transitions and animations to enhance user experience.
